# VULKANO — піцерія на дровах

Premium demo website for a wood-fired pizza delivery brand. **v2: Next.js 15 + GSAP + Lenis + OGL WebGL.**

**Live:** https://btw-hash.github.io/vulkano-pizza/

## Features

- Multi-page: home, filterable menu, pizza configurator (size / crust / extras), cart + checkout with validation, animated order tracker, about.
- Working cart on `localStorage`: slide-in drawer, quantity controls, promo code `VULKANO10`, free-delivery threshold.
- Simulated checkout with animated success state and live order tracker (progress line, ETA countdown, replay).
- Motion design v2: oven-door opening wipe, SplitText char-ignition headline, WebGL heat-haze over hero and craft imagery, pinned "oven journey" scroll scene with live temperature counter, ember particles, ember-spark cursor with click flare, add-to-cart flight along an arc, view-transition circle wipes between pages, flame-burst success, glowing tracker. Respects `prefers-reduced-motion`.
- Fully responsive (390px → 1440px), Ukrainian language, Cyrillic display typography (Unbounded + Golos Text).

## Stack

Next.js 15 (static export) · GSAP + ScrollTrigger + SplitText + Flip · Lenis · OGL (heat-haze shader) · next/font · Unsplash photography. Design & code: BTW Studio, 2026. Demo project — all brand names fictional.
